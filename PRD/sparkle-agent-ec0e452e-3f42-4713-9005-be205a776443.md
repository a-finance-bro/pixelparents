## Progress Update as of June 28, 2026 — 3:12 PM Pacific

### Summary of changes since last update
Large signup-flow + landing redesign. Added a live, animated grayscale Voronoi
"interest mosaic" background (interlocking polygon tiles that flip through the
real interest list) on the home, /signup, and /builders pages. Reworked the
home page into a sign-up CTA with live counts. Restructured the signup flow so
the step-1 parent form now collects location (city/state), parent interests, and
family photos, and the step-2 thanks page is just children. Added a required
"are you a builder?" question (builder / aspiring / no) stored in signups.extra,
contextual helper panels, a co-parent invite UI scaffold's sibling copy, and an
admin "Builder?" column. Many copy tweaks.

### Detail of changes made:
- New `app/signup/interest-tiles.tsx`: client Voronoi mosaic (deterministic seeds
  → Sutherland–Hodgman half-plane clipping → convex polygon tiles). Variants:
  "strip" (signup/builders top strip), "fade" (home, fades to black mid-page),
  "full". Tiles flip to random interests over time; SSR-safe (no Math.random at
  render).
- New `lib/interest-icons.ts`: word/stem-based interest→Lucide icon mapping
  (~90 groups). Added `lucide-react` dependency.
- New `components/pixel-mascot.tsx`: shared bordered mascot (3px gray border, 2px
  gap), used on home/signup/builders.
- Home (`app/page.tsx`): async, live counts; "[Sign up] to Join N…" + connect
  line + IRL notes; footer "Created with ❤️ by N technical + N non-technical
  parents… [Learn more]" → /builders.
- `lib/db/signups.ts`: getSignupCount/getChildrenCount/getInterestsCount/
  getBuilderCounts helpers.
- Parent form (`app/signup/signup-form.tsx`): required builder question + helper
  panels; student-resource opt-in panel (on LinkedIn); conditional section
  visibility by builder answer; City/State; parent interests; family photos
  (shared TagPicker/PhotoUploader exported from family-form); button "Add Your
  Child(ren) →".
- Thanks page (`app/signup/thanks/*`): removed personal intro block; heading
  "Tell us about your child(ren)"; step 2 is children only; share settings copy
  reworked (OHS directory framing, /directory placeholder link, all share fields
  default on).
- Admin: "Builder?" column (Yes: Technical / Yes: Curious / No).
